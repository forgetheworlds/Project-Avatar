## OscarLiang: Nano Long Range 3" Build (1S 18650) — Lessons from Failure
- **URL:** https://oscarliang.com/nano-long-range-build/
- **What:** Attempted nano long-range build with 1S 18650 Li-Ion. Multiple failures, detailed root cause analysis, invaluable lessons for micro builds.
- **Platform:** 1S AIO FC, 1202.5 motors, 18650 Li-Ion
- **Cost:** ~$100 (attempted)
- **Success/Failure:** FAILURE — ESC bricked, FC unable to handle voltage sag. Recommended BNF (Rekon 3) instead.
- **Key Lesson for Avatar:** THREE CRITICAL LESSONS: (1) Battery contact resistance can kill a build — solder directly, don't use spring contacts. (2) Not all FCs handle Li-Ion voltage sag — test the ESP32's brownout behavior with sagging 1S. If using Li-Ion for long cinematic flights, we need a boost regulator between battery and ESP32. (3) Always use a dedicated 5V step-up for VTX/camera — FC 5V rail is too weak.
- **Priority:** NOW

## 🎯 KEY ARCHITECTURAL INSIGHT

After surveying 30+ projects, the optimal Avatar architecture is clear:

```
ESP32-S3 (Companion Computer)
  ├── PMW3901 Optical Flow → horizontal velocity
  ├── VL53L1X ToF → altitude 
  ├── TinyGPS → global position
```
