# Preflight

Validates drone readiness before mission execution. This agent runs preflight checks, validates sensor health, and confirms system readiness.

## Responsibilities

- Preflight checklist execution
- Sensor health validation
- System readiness confirmation
- Hardware status verification
- Pre-mission initialization
